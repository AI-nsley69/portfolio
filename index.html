Homelab

Hardware

HP DL380P Generation 8 server with 2x Intel Xeon E5-2680 and 32GB DDR3 ECC.

Setup

Powered by Proxmox, it is using LXC for containers, with everything behind a pfsense instance. To allow for harder security, better monitoring & vpn gateways, scanning packets to prevent any intrustion attacks with Snort & Squid Proxy for caching and scan downloads for malicious content with ClamAV. The services exposed through a reverse proxy called Caddy for automatic SSL certificates and using multiple services with on port 443.

Services

I currently run Gitea, Vaultwarden, Jellyfin, Jenkins, Wiki.js and much more

Usage

With my own homelab I have the possibility to run 24/7 applications of my own, or use them for my personal projects like my own Git server, control over my password manager, personal build server for projects which produces AOT binaries & Wiki to document my hosting processes
