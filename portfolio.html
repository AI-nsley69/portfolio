AinsBOT

A personal Discord Bot written in Node.js, some features such as fetching tiktoks, reddit posts & message links to display their content. More fun commands like fetching animals, interacting with users through gifs (hugs etc) and the ability to disable most commands/features!

Khaos Utils

A Discord Bot written in Node.js to serve the community of a Technical Minecraft Server managed by me. Features include application management (creating valid application links through secrets, setting up polls for apps, creating a channel for the applicant etc), communicating with the Minecraft servers (gathering status on proxy & the 3 minecraft servers and bridging all of the chats together with a websocket server and server-side mods) moderation commands (kick, ban, mute etc), & more!

imbed

A small program to embed any data into an image written in Zig. It is achieved through appending the data after the JPG header to end the image, this causes no corruption/change in the image aside from its storage size. Currently requires some finicky methods to embed anything other than a txt file (i.e rename file to message.txt, embed that then when reversing the process revert the name). Encrypts the data through XSalsa20Poly1305 through user input password.

MCRISC1608

A RISC-inspired Minecraft CPU built with Redstone. Started as a learning project with TheWildJarvi as a mentor. 8-bit CPU with 7 registers & 32 bytes of RAM. Had access to 3x 7-segment displays (8-bit) and a mod 32 Matrix display. Was able to sort 5 numbers through bubble sort in 20 minutes.

ZigF*ck

A small Brainf*ck interpreter written in Zig. Iterates over all characters in a given file, however, it discards any non-brainf*ck characters, then sends them through a switch case to interpret each and every one of them.

Brainf*ckToZig

A small Brainf*ck transpilerto & in Zig. Uses code from the before mentioned interpreter to append to a string. Uses the built-in standard library tool to parse it. Main advantage over interpreter is the speed up benefit from the Zig compiler

ZigBrot

A simple (multithreaded) Mandelbrot renderer written in Zig. Was intended on rendering multiple frames for a zooming video. Written using the pseudocode found on Wikipedia as a reference.

True All Of Fabric

A small program written in Python to gather all of the fabric mods from a current mc version and downloading them into a single directory. Created as a joke in response to a modpack known as All Of Fabric which is not literal. However, this program is!

MonoPew

A small space invaders-like game created in C# with the MonoGame framework as a school assignment. All game art was created in under 5 minutes through Microsoft Paint as the main focus was the game itself. Supports multiple levels where each level increases speed for both yourself & the enemies, however, you need to kill more enemies to spawn in the boss.

Luckperms Group Whitelist

A small Velocity plugin in Java, which depends on a permission-based plugin to only allow certain groups to join specific servers on the proxy. Created with the intention of using it on Minecraft networks to minimize the needed server-side mods & manual management for whitelist.

Repl API

A nodejs express app for my own personal API usages. Currently in use with AinsBOT for fetching information about characters from a mobile game with unofficial resources due to lack of one. Optimized with caching and reducing external requests if possible.
