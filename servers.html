Homelab

My homelab is a HP DL380P Generation 8 server powered by 2x Intel Xeon E5-2680 and 32GB DDR3 ECC. Due to some issues with the consumer harddrives on it, the iLO firmware is hacked to allow for fancontrol through a script I made myself to keep the fans low due to the environment it is in. Powered by Proxmox as the Operating System, it is using LXC to containerize as many services as possible, with all containers running in their own subnet behind pfsense. This lets me tighten the forwarding rules, use vpn as gateways for certain services that I do not want the request to come from my public ip, scanning packets to prevent any intrustion attacks with Snort & Squid Proxy to cache requests and scan downloads for malicious content with ClamAV. The services are not exposed directly, but through a reverse proxy done with Caddy for automatic SSL certificates and using multiple services with one port (443 specifically) through subdomains. Some services include my own git server with the Gitea application, my own Bitwarden instance with Vaultwarden as the implementation for the backend, Jellyfin as a media for all of my media needs, Jenkins as build server to compile my own software and other self-hosted software like Vaultwarden and my own wiki with Wiki.js to document the majority of things I do and how I setup stuff.
